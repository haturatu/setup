# setup
My personal setup script for a new computer.

# Usage
Clone this repository and run the `setup.sh` script:

```bash
git clone https://github.com/haturatu/setup.git
cd src
chmod +x setup.sh
./setup.sh
```
## Support
### apt
debian, ubuntu, linuxmint, devuan

### pacman
arch, manjaro, artix

### homebrew
maybe ...
